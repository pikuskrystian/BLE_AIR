## General Info

The goal of the project is to write Qt Aplication which can connect to our embedded system,
and read sensor information from it. Which will be displayed in real time.
## Technologies

Project is created with:
* QML
* QMake
* C++

## Hardware
* NXP LPC55S69
* BMP 280 sensor
